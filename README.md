**Bundle ID:** app.fairvalueauto

## 🚀 Quick Setup (Run These Commands In Order)

### Prerequisites
Before starting, make sure you have these installed:

```bash
# Check if Node.js is installed (need v18+)
node --version

# Check if npm is installed
npm --version

# Check if Git is installed
git --version
```

If Node.js is missing: Download from https://nodejs.org (choose LTS version)

---

### Step 1 — Install Dependencies
```bash
npm install
```

### Step 2 — Install Capacitor CLI globally
```bash
npm install -g @capacitor/cli
```

### Step 3 — Add iOS platform (Mac only)
```bash
npx cap add ios
npx cap sync ios
```

### Step 4 — Add Android platform
```bash
npx cap add android
npx cap sync android
```

### Step 5 — Open in Xcode (iOS, Mac only)
```bash
npx cap open ios
```
Then in Xcode: Product → Archive → Distribute App → App Store Connect

### Step 6 — Open in Android Studio
```bash
npx cap open android
```
Then in Android Studio: Build → Generate Signed Bundle/APK → Android App Bundle

## 🔧 Updating the App

When you make changes in Base44, run:
```bash
npx cap sync
```
This syncs the latest version from the live URL to both platforms.

---

## 💡 Troubleshooting

**"npx cap" not found** → Run: `npm install -g @capacitor/cli`  
**iOS build fails** → Ensure you're on a Mac with Xcode 15+  
**Android build fails** → Ensure Android Studio is installed with SDK 34  
**App shows blank screen** → Check that `velox-satisfied-deal-flow.base44.app` is live
